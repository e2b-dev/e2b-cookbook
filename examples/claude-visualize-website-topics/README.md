# Visualizing Website Topics

Powered by [Claude 3.5 Sonnet](https://www.anthropic.com/news/claude-3-5-sonnet), [Firecrawl](https://www.firecrawl.dev/), and [Code Interpreter SDK](https://github.com/e2b-dev/code-interpreter) by [E2B](https://e2b.dev/docs)

This project scrapes a website using Firecrawl, then uses Claude AI to analyze and visualize the most common topics found in the content using E2B's Code Interpreter.

## Features

- Web scraping with Firecrawl
- AI-powered topic analysis using Claude 3.5 Sonnet
- Code execution and visualization in secure sandbox using E2B
- Automatic topic extraction and visualization

## Prerequisites

- Python 3.10 or higher
- Poetry package manager
- API keys for:
  - [Anthropic Claude](https://console.anthropic.com/)
  - [Firecrawl](https://www.firecrawl.dev)
  - [E2B](https://e2b.dev/docs)

## Installation

1. Install Poetry if you haven't already:
```bash
curl -sSL https://install.python-poetry.org | python3 -
```

2. Clone or navigate to this project directory

3. Install dependencies using Poetry:
```bash
poetry install
```

4. Copy the example environment file and add your API keys:
```bash
cp .env.example .env
```

5. Edit `.env` and add your API keys:
```
ANTHROPIC_API_KEY=your_actual_anthropic_api_key
FIRECRAWL_API_KEY=your_actual_firecrawl_api_key
E2B_API_KEY=your_actual_e2b_api_key
```

## Usage

Run the project using Poetry:

```bash
poetry run run
```

Or activate the Poetry shell and run directly:

```bash
poetry shell
python -m src.main
```

## How It Works

1. **Web Scraping**: The application uses Firecrawl to crawl a specified website (default: LangChain documentation) and extract content from up to 5 pages.

2. **Topic Analysis**: Claude AI receives the crawled content and is instructed to identify common topics using Python code.

3. **Code Execution**: The Python code generated by Claude runs in E2B's secure Code Interpreter sandbox.

4. **Visualization**: The results are visualized as a bar chart showing the frequency of each topic.

## Customization

You can modify the target URL in `src/main.py`:

```python
crawl_url = 'https://your-target-website.com'
cleaned_crawl_result = crawl_website(crawl_url, limit=5)
```

Change the `limit` parameter to crawl more or fewer pages.
